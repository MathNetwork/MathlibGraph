# MathlibGraph

Code and data for analyzing the dependency structure of [Mathlib](https://github.com/leanprover-community/mathlib4), the largest unified library of formalized mathematics.

## Quick Start

Install dependencies and run basic statistics:

```bash
pip install datasets networkx pandas numpy matplotlib powerlaw python-louvain scikit-learn
```

```python
from datasets import load_dataset

nodes = load_dataset("MathNetwork/MathlibGraph",
                     data_files="mathlib_nodes.csv", split="train").to_pandas()
edges = load_dataset("MathNetwork/MathlibGraph",
                     data_files="mathlib_edges.csv", split="train").to_pandas()

nodes = nodes.drop_duplicates(subset="name", keep="first")
print(f"Declarations: {len(nodes):,}, Edges: {len(edges):,}")
# Declarations: 308,129, Edges: 8,436,366
```

## Three Graph Levels

The analysis operates on three progressively finer dependency graphs:

| Graph | Nodes | Edges | Granularity |
|-------|-------|-------|-------------|
| `G_module` | 7,563 modules | 23,570 imports | Source file dependencies |
| `G_thm` | 308,129 declarations | 8,436,366 premises | Theorem/definition dependencies |
| `G_ns^(k)` | ~6,300 namespaces (k=2) | ~92,000 edges | Namespace-aggregated dependencies |
